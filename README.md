# AI Chatbot Assistant

An AI-powered chatbot that answers queries related to college or company information using natural language processing.

## 🚀 Features
- Real-time chat interface
- Context-aware responses using AI
- FAQ-based knowledge system
- Scalable backend for handling queries

## 🛠 Tech Stack
- Frontend: React.js
- Backend: Node.js, Express
- AI Integration: OpenAI API / NLP
- Data: JSON / Knowledge base

## 📂 Project Structure
frontend/ → UI  
backend/ → APIs and AI logic  
data/ → FAQs and documents  

## ⚙️ Setup Instructions

### Clone repo
git clone https://github.com/Gautam-Rawat/ai-chatbot-assistant.git

### Install dependencies
cd frontend && npm install  
cd ../backend && npm install  

### Run project
cd backend && npm start  
cd ../frontend && npm start
